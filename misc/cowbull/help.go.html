Find the hidden word using the position clues. See sample game
  • Hidden word has four letters, no repeats
  • Enter guesses, four letters, no repeats
  • Guesses have to be valid English words
  • For each guess, the number of matching letters is shown
  • Exact positional match is a BULL (letter is same position in the guess and in the hidden word), like in bulls-eye ðŸŽ¯
  • Partial match (letter is present in the hidden word but at a different position) is a COW

For example, if the hidden word is WORK,
  • Guess ROSE has 1 bull (O), 1 cow (R)
  • Guess CORD has 2 bulls (O & R), 0 cows
  • Guess WORK has 4 bulls, game over

Tips

  • Single-player, play by yourself, try to guess the hidden word
  • Setup a multi-player game, share challenge link with others to have then join. All players play with the same word
  • Game clock starts after the first guess is entered
  • Game score depends on the number of clues used and time taken. Max score is 10
